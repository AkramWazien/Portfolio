Title: DIY alarm clock

Date: 18-07-2026

This project stems from my issue with phone distraction during study sessions. This alarm clock is set to strictly 20 minutes for ease of use purpose and cause that is how long I can stay in deep focus before my focus reced. It also let my eye rest from screen for a few minutes. After 20 minutes, the buzzer will make a sound and it resets back to 20 minutes. There is also a clock function that tells me the time in case I lose track of time.

From this project, I learned to use the RTC clock module as well as LCD module to make a difference in my life.
