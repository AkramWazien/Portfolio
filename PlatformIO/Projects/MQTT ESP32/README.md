Title: ESP32 MQTT protocol

Date: 21-07-2026

This project covers the common communication protocol of IoT devices, MQTT. In this project, I built a publisher that send a message to the broker. It is also subscribed to another domain of the broker and can receive message from the subscriptor. In the future, I may integrate this project with my SPIFFS project to ensure the data is sent and saved and I may add a command receiver on the ESP32.

From this project, I learned about IoT devices protocol and was able to use WiFi to send data.
