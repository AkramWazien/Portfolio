# Title: ESP32 Web Controller

Date: I forgot

This project is my attempt at trying to control a component through WiFi. It is certainly hard since I don't really know much about HTML and that is the reason I created the HTML folder. From this project, I realised that I need to have a bit of knowledge about HTML. I was able to learn about HTML and understand every bit of my code. My networking knowledge also helped too. Essentially, everytime a button in the web is pressed, it will send a GET request to the server which is ESP32 and it will respond with a POST method that tells the client where to go.

From this project, I was able to learn about GET and POST methods as well as understanding ESP32 WiFi capabilities.
